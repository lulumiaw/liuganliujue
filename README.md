# 🌟 六感六覺作文練習遊戲

適合國小低年級使用的作文練習遊戲，透過六感六覺（視覺、聽覺、嗅覺、味覺、觸覺、心覺）引導孩子組成完整句子。

## 遊戲說明

1. 畫面出現隨機圖片題目（動物、植物、自然、人物、日常物品）
2. 選兩種感覺（五種感覺中選兩種）＋必選心覺
3. 每種感覺可選 1–3 個語詞（含注音）
4. 系統自動套入句子框架，產生 4 種句型供切換
5. 完成後語音朗讀句子，並評估語詞是否適合

## 功能特色

- ✅ 14 個隨機題目
- ✅ 推薦語詞（黃色）與挑戰語詞（灰色）區分
- ✅ 環境描述詞（藍色）說明
- ✅ 語音輸入（需 Chrome）
- ✅ TTS 語音朗讀完成句子
- ✅ 孩子可自行新增語詞
- ✅ 完成後評估語詞適切性並提供改寫建議

## 使用方式

直接用 Chrome 瀏覽器開啟 `index.html`，或透過 GitHub Pages 連結使用。

## 技術說明

純前端實作，不需要伺服器，使用 Web Speech API 進行語音輸入與朗讀。
